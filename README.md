# PortfolioHUB

Site estático para GitHub Pages que apresenta o perfil de Bernardo e lista automaticamente repositórios públicos do GitHub.

## Tecnologias

- HTML semântico
- CSS responsivo
- JavaScript com Fetch API
- GitHub REST API

## Publicação

O site pode ser publicado diretamente pelo GitHub Pages a partir da branch `main`, usando a pasta raiz do repositório.
